# flower-species-recognition

Flower species recognition on Tensorflow.

## 🛒 Requires

- Python3
- PIP3

## 🍽 Usage

```python
python -m scripts.label_image \
    --graph=tf_files/retrained_graph.pb  \
    --image=tf_files/flower_photos/roses/2414954629_3708a1a04d.jpg 
```
